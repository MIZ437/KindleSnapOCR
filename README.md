# KindleSnapOCR

Kindle本を画面キャプチャでPDF化するWindowsツール。私的利用目的専用。

## 機能

- 指定領域の自動スクリーンキャプチャ
- 自動ページ送り（左右対応）
- 最終ページ自動検出
- PDF生成
- OCR処理（Tesseract、オプション）
- プライバシーモード（キャプチャ領域を黒で隠す）

## 使い方

1. `start.bat` をダブルクリック
2. 書籍タイトルを入力
3. 「キャプチャ範囲を選択」でKindleの本文領域を選択
4. ページ送り方向を選択（横書き→右、縦書き→左）
5. 「キャプチャ開始」をクリック
6. ESCキーで中断可能

## 必要環境

- Windows 10/11
- Python 3.8以上

## インストール

初回起動時に自動でセットアップされます。

## OCR機能（オプション）

OCR機能を使用するには、別途Tesseractのインストールが必要です。
