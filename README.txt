================================================================================
KindleSnapOCR - Kindle本PDF化ツール
================================================================================

【使い方】

1. start.bat をダブルクリック
   (初回は自動でセットアップされます)

2. Kindleアプリで本を開く

3. 設定を行う
   - 書籍タイトルを入力
   - 「キャプチャ範囲を選択」で本の表示領域を指定
   - ページ送り方向を選択（縦書き→左、横書き→右）

4. 「キャプチャ開始」をクリック
   - 5秒後に自動キャプチャ開始
   - 処理中はPCを触らない
   - 中断したい場合はESCキー

5. 完了後、ドキュメント\KindleSnapOCR にPDFとOCRテキストが保存されます

================================================================================

【必要環境】
- Windows 10/11
- Python 3.8以上 (https://www.python.org/downloads/)
- Kindle for PC

※ OCR機能はWindows標準機能を使用するため、追加インストール不要です

================================================================================

【ポータブル版の作成】
build.bat を実行 → dist\KindleSnapOCR\ に出力されます
